# addBody

```js
l1.addBody(entity, body)
```

Add a physics body from Matter.js to an entity. Is auto-curried.

## Arguments

`body` (Matter.body): A Matter body.

`entity` (object): The entity to apply body to.

## Returns

(Object): The created body.
